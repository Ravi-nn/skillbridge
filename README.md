# SkillBridge 🚀
**Find your skill gap. Get hired faster.**

SkillBridge is a free web app that compares your current skills against real job demand in your city — and gives you a step-by-step learning roadmap to close the gap.

🔗 **[Live Demo](https://ravi-nn.github.io/skillbridge)**

---

## What It Does

1. You enter your city and current skills
2. It scans real job listings in your area
3. It shows exactly which skills you're missing and a free course for each one
4. You get a Job Readiness Score so you can track your progress

---

## Features

- 📡 Real job data via Adzuna API (free API key)
- 📊 Animated job readiness score
- 📚 Free learning resources for every missing skill
- 🏢 Live job listings you can apply to
- 📤 Share your skill report
- 📱 Works on mobile and desktop
- 🔑 API keys stored locally — never sent anywhere

---

## Tech Stack

- HTML, CSS, JavaScript (no frameworks, no build tools)
- Adzuna Jobs API
- GitHub Pages (free hosting)

---

## Run Locally

```bash
git clone https://github.com/Ravi-nn/skillbridge.git
cd skillbridge
open index.html
```

No setup needed. Just open the file in your browser.

---

## Connect Real Job Data (Optional)

1. Get a free API key at [developer.adzuna.com](https://developer.adzuna.com/)
2. Open the app and enter your App ID and App Key
3. That's it — live job listings are now active

---

## Deploy to GitHub Pages

```bash
git init
git add .
git commit -m "Launch SkillBridge"
git branch -M main
git remote add origin https://github.com/Ravi-nn/skillbridge.git
git push -u origin main
```

Then go to **Settings → Pages → Source: main branch** and your app is live.
